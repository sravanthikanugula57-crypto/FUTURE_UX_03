# FUTURE_UX_03

## CRM Dashboard UI

### Project Overview

This project is a CRM (Client Relationship Management) Dashboard UI designed for a digital marketing agency as part of the Future Interns UI/UX Design Internship.

### Objective

To help agencies manage leads, clients, follow-ups, and tasks efficiently through a clean and scalable dashboard interface.

### Screens Included

1. Dashboard Overview
2. Lead Pipeline
3. Client Profile
4. Tasks & Follow-ups

### Features

* Lead management
* Client tracking
* Visual sales pipeline
* Task and follow-up management
* Analytics overview
* Professional SaaS dashboard design

* Figma
